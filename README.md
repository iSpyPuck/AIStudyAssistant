# AI Study Assistant (Python)

This is a simple AI-powered study assistant written in Python.  
It reads its behavior/personality from `behavior.txt`, uses OpenAI models,  
and keeps API keys secure using environment variables.

---

## Features

- Loads its instructions from `behavior.txt`
- Works in your terminal as a chat interface
- Uses environment variables to protect API keys
- Easy to customize and extend
